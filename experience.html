  • Holo-Desk with the SSE

    Holo-Desk was a student-run project by the Society of Software Engineers (SSE) that aimed to create an interactive tabletop system. During the spring of 2016, I helped work on the system with the SSE in order to get it ready for display at Imagine RIT, an annual festival showcasing innovative student projects. Individually, I collaborated with another member to write code that would enable accelerometers to detect taps on a tabletop. Once taps were detected, a Raspberry Pi would then be notified with all of the appropriate data. We utilized the following technologies:

    • Python
    • C++
    • Raspberry Pi
    • Arduino
    • Kinect
    • Accelerometer
