  • Healthcare Management (Academic Project)
    Led a group of 3 students to develop a website for a fictitious hospital to be used by healthcare staff and patients as an academic project, employing a variety of technologies.

    more

    less

    • Django

    HealthNet was a semester-long project assigned to software engineering students in Intro to Software Engineering. The class was divided into groups of 4 students; I became the team leader of my group. We were tasked with developing a website that would simulate the behavior of a hospital management system.

    We were responsible for handling the documentation, planning, construction and deployment of the project, which was broken up into two releases. Our first release received a grade of 82%, which wasn’t to our satisfaction. We quickly made improvements to our process so that we wouldn’t repeat the same mistakes. Our second release received a grade of 92%, tied with one other group for the highest grade in the class.

    To complete this project, we were required to use Python and HTML. We also had to learn how to use Django, a web framework for Python, and PyCharm, an IDE. To handle the CSS of our website, we turned to Material Design Lite, a CSS framework. We practiced the agile software development process throughout the project.

    Along with leading and organizing my team, I was personally responsible for implementing client-side form validation with jQuery, creating dynamic HTML pages with Django and implementing numerous project requirements.

    Holo-Desk was a student-run project by the Society of Software Engineers. It was an attempt to make a plug-n-play touch display that can be used on any surface, especially tabletops. The initial prototype made use of Raspberry Pis, Arduinos, projectors, a Microsoft Kinect and accelerometers.

    My responsibilities included collaborating with another team member to write code that would allow the accelerometers to communicate with a Raspberry Pi so that taps on a surface could be detected. In addition, I gained a lot of experience working with the Arduino Mega and Arduino Leonardo through reviewing and improving the code written by other team members.
